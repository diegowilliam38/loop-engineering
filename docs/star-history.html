GitHub access token

Star history reads from the GitHub API. Since mid-2026, stargazer data is limited to repos you own or collaborate on — and requires a scoped token.

Create a fine-grained token with only Metadata → Read-only, or a classic token with public_repo. More details.

Tokens with no scopes no longer work, even for your own repos.

  • Fine-grained: Repository access → this repo. Permissions → Metadata → Read-only.
  • Classic: tick public_repo.

Never sent to Loop Engineering servers — chart requests go directly to GitHub.
